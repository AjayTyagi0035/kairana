Place your shapefile components in zip files in the respective layer folders:

/public/maps/layer1/
  - layer1.zip (containing: layer1.shp, layer1.shx, layer1.dbf, layer1.prj, layer1.cpg)

/public/maps/layer2/
  - layer2.zip (containing: layer2.shp, layer2.shx, layer2.dbf, layer2.prj, layer2.cpg)

/public/maps/layer3/
  - layer3.zip (containing: layer3.shp, layer3.shx, layer3.dbf, layer3.prj, layer3.cpg)

/public/maps/layer4/
  - layer4.zip (containing: layer4.shp, layer4.shx, layer4.dbf, layer4.prj, layer4.cpg)

Note: 
- Create a zip file containing all shapefile components
- The .shp and .dbf files are required
- The .shx, .prj, and .cpg files are optional but recommended
- All files should have the same base name (e.g., layer1.shp, layer1.dbf, etc.)
